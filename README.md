# Staff Database Forms

Приложение на C# Windows Forms для работы с базой данных сотрудников (Access).

## Функциональность

### Главная форма (Form1)
- Просмотр таблицы сотрудников через DataGridView
- Удаление сотрудника по коду
- Открытие формы добавления нового сотрудника
- Обновление данных

### Форма добавления (Form2)
- Ввод данных нового сотрудника:
  - Код сотрудника
  - Фамилия
  - Имя
  - Отчество
  - Код должности
- Добавление записи в базу данных

## Интерфейс

### Form1
- DataGridView — отображение таблицы сотрудников
- TextBox — ввод кода для удаления
- Кнопка "Удалить" — удаление по коду
- Кнопка "Добавить" — открытие Form2
- Кнопка "Обновить" — обновление данных

### Form2
- Поля ввода для всех атрибутов сотрудника
- Кнопка "Добавить" — сохранение в базу

## Куда можно встроить

- В кадровые системы учёта
- В учебные проекты по базам данных
- Как пример работы с OleDb и Access
- В CRM-системы малого бизнеса

## Требования

- Windows
- .NET Framework / .NET Core с WinForms
- Microsoft Access Database Engine (ACE)
- Файл базы Кадровый_состав.accdb в папке с программой

## Примечание

Подключение использует провайдер Microsoft.ACE.OLEDB.12.0. Для работы требуется установленный драйвер.
